Що я хочу дізнатись?: Що таке Map Set і як ними користуватись

Що я дізнався?:
Map це структура даних в js, яка складається з пар ключ значення.
Відмінності від об'єкта:
-Ключем може бути що завгодно
-Є додаткова властивість size

Set це структура даних в js, яка складається тільки зі значень. Значення всі унікальні. При добавлянні значення в сет він автоматично первіряє чи є це значення у ньому.
-Методи всі ті самі що і у мап.

Як я це буду застосовувати в майбутньому?: Наприклад для запису унікальних відвідувачів. Для зберігання за кожним юзером якогось значення.
